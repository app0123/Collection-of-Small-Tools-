# Collection-of-Small-Tools-
專門為低效能、低運作能力者提供高實用度的小工具

目前已公布實用小工具:
1.聲音播放器

聲音播放器

用處:
專門播放各種音訊，並提供各種相關功能，且能夠直接抓取要撥放單一內容至命令提示字元內

實用內容:
提供100％完整撥放，不任意刪除或減少撥放清單數量，並完全支援離線撥放(下載重要檔案除外)

本小工具特點:
超低效能、單一視窗、只需安裝Python 3.13.12版本與可用命令提示字元、不大量占用記憶體(不超過50.0 MB)與硬體空間(不超過20 KB)(請注意:同意下載FFmpeg擴充後，可能需要百KB空間)、高實用度

支援撥放格式:
普通版解析:.mp3、.wav、.flac、.ogg、.opus
進階版解析:.webm、.m4a / .alac、.aac、.ape、.wv、.mp4
(普通版與進階版差異點只因為是否同意自動下載FFmpeg擴充)

如何使用:
1.啟動命令提示字元
2.打入指令(下載位置可訂)        curl -o python-3.13.12-amd64.exe https://python.org
3.打入指令(必須導引下載後位置)   python-3.13.12-amd64.exe
注意!!!:啟動安裝畫面後，請務必勾選最下方的 「Add python.exe to PATH」
4.下載完畢後，請把cd 您要執行的檔案位置
5.打入  py 音樂撥放器.py
