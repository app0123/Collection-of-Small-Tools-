# Collection-of-Small-Tools (實用小工具合集)
專門為低效能、低運算能力的電腦環境，提供高實用度的小工具。

## 目前已發布工具：終端機純音訊播放器

**專案理念：**
專門在命令提示字元 (CMD) 中運作的極簡音樂播放器，提供快捷鍵操作與進度條，並能直接拖曳檔案或資料夾進行播放。

**核心特色：**
* **極致輕量：** 單一視窗運作，主程式腳本極小（不超過 20 KB），執行時記憶體佔用極低（原生播放通常不超過 50 MB）。
* **原汁原味：** 100% 完整讀取播放清單，不漏歌、不擅自刪減。
* **離線支援：** 完全支援無網路環境離線播放（初次啟動若需下載 FFmpeg 擴充除外）。

**支援播放格式：**
* **普通版解析 (內建原生支援)：** `.mp3`, `.wav`, `.flac`, `.ogg`, `.opus`
* **進階版解析 (需同意自動下載 FFmpeg 擴充)：** `.webm`, `.m4a`, `.alac`, `.aac`, `.ape`, `.wv`, `.mp4`
*(註：普通版與進階版的差異，僅在於首次啟動時是否同意讓程式自動下載 FFmpeg 解碼核心。)*

## 如何安裝與使用

**步驟 1：安裝 Python 環境**
若您的電腦尚未安裝 Python，請至 [Python 官方網站](https://www.python.org/downloads/) 下載最新的 Windows 安裝檔 (建議 Python 3.8 以上版本，或直接尋找Python 3.13.12版本)。
> **重要提示：** 在執行安裝程式時，請務必勾選畫面最下方的 **「Add python.exe to PATH」**，否則終端機將無法辨識指令

**步驟 2：開啟命令提示字元並切換目錄**
開啟「命令提示字元 (CMD)」，並使用 `cd` 指令切換到您存放小工具的資料夾位置。例如：  
`cd C:\Users\您的名稱\Downloads\Collection-of-Small-Tools-`

**步驟 3：執行播放器**
輸入以下指令即可啟動程式：
`py 音樂播放器.py`
*(啟動後，程式會自動檢查並引導您安裝必要的輕量套件，請依畫面提示操作即可開始聽歌！)*
